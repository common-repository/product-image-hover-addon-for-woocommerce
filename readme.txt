Product Image Hover Addon For WooCommerce is a custom plugin for WooCommerce.

== Description ==
Plugin helps to add an additional image for WooCommerce Product. 
The image will visible on mouse hover product featured image.

Requirments:
* WooCommerce must be active plugin.

== Installation ==

1. Upload the entire `woo-product-image-hover` folder to the `/wp-content/plugins/` directory.
1. Kindly make sure 'WooCommerce' plugin active before activate this plugin.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Edit the product and upload the additional image.
